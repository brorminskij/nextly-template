# Tranquility - Private Members Club

Tranquility is a private members-only club template site focused on wellbeing,
curated events, and a respectful community. This repo contains the website
for the Tranquility club built with Next.js & TailwindCSS.
 
![image](https://user-images.githubusercontent.com/1884712/121497169-03228680-c990-11eb-975a-e77fddc43de0.png)

## How to use

Use this repository as the Tranquility website. Clone the repo and follow the quick start steps below.
### Quick start

1. Clone the repository

```powershell
git clone <your-repo-url>
cd tranquility-site
```

2. Install dependencies

```powershell
npm install
# or
yarn install
```

3. Run dev server

```powershell
npm run dev
```
